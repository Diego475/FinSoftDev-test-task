# FinSoftDev

## Запуск всех пакетов
```
composer i
````
## Запуск
Для запуска ввести команду:
```
php -S localhost:8000
```

## Как это работает?
После запуска будет доступен единственный маршрут на `http://localhost:8000/api/v1/future`
